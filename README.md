# ⚡ BrieflyAI — AI News Summarizer

## 🚀 Overview

BrieflyAI is a modern AI-powered web app that delivers **instant news summaries** using real-time search and LLMs.

Just enter a topic → get clean, concise bullet-point insights.

---

## ✨ Features

* 🔍 Real-time news search (Tavily)
* 🧠 AI summarization using Mistral
* ⚡ Fast and responsive UI
* 🎨 Glassmorphism + neon styled interface
* 📊 Clean bullet-point outputs

---

## 🛠️ Tech Stack

* Python
* Streamlit
* LangChain
* Mistral AI
* Tavily API

---

## ⚙️ Setup Instructions

### 1. Clone the repo

```bash
git clone https://github.com/your-username/briefly-ai.git
cd briefly-ai
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Setup `.env`

```
TAVILY_API_KEY=your_key
```

---

## ▶️ Run the App

```bash
streamlit run BrieflyAI.py
```

## 🧠 How It Works

1. User enters query
2. Tavily fetches real-time news
3. Mistral AI summarizes it
4. Output displayed in clean UI
